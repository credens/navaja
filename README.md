# NovaPanes Website

Este es el sitio oficial de NovaPanes. Usa Vite + Express + Tailwind + TypeScript + React.

---

## 🚀 Cómo correr el sitio localmente

### 1. Cloná el repositorio

```bash
git clone https://github.com/tu-usuario/novapanes-website.git
cd novapanes-website
```

### 2. Instalá las dependencias

```bash
npm install
```

### 3. Modo desarrollo (recomendado para trabajar en local)

```bash
npm run dev
```

> Esto levanta el backend desde `server/index.ts`.

### 4. Modo producción (build + run)

```bash
npm run build
npm start
```

---

## 🛠️ Requisitos

- Node.js >= 18
- npm >= 9
